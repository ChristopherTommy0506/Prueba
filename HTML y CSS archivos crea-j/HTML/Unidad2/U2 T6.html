Tema 6 :Concepto de tamaño.

Imagen de: Peques aprenden jugando

El tamaño se refiere a qué tan pequeño o grande resulta un objeto físico.


Grande

El tamaño grando singnifica que el objeto tiene un tamaño mayor al normal como por ejemplo una mansion.

Imagen de: dibujos.net

Mediano

La palabra mediano singnifica que el objeto tiene un tamaño normal, ni muy grande ni muy pequeño.

Imagen de: blendspace

pequeño

Un tamaño pequeño significa que tiene un tamaño menor de lo normal, como por ejemplo un perrito.

Imagen de: Animal mascota.

Actividad

Busca 3 objetos en tu casa y diferencia cual de los 3 es el grande, el mediano y el pequeño.
